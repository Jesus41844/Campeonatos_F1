# F1 Championships Dashboard

Dashboard web para ver la clasificación de pilotos de Fórmula 1 por año.  
Proyecto base para practicar consumo de APIs y luego integrar datos IoT.

---

## Funcionalidades
- Consulta la clasificación de pilotos por temporada
- Permite escribir el año manualmente
- Consume una API pública de F1
- Interfaz simple con estilo inspirado en F1

---

## Tecnologías
- HTML
- CSS
- JavaScript
- Ergast F1 API (jolpi.ca)

---

## API
- Endpoint:
  https://api.jolpi.ca/ergast/f1/{year}/driverStandings.json

- Ejemplo:
  https://api.jolpi.ca/ergast/f1/2023/driverStandings.json
